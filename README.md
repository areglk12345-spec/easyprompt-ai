# 🌟 EZPrompt AI (Accessibility & Prompt Fitter Agent)

**EZPrompt AI** คือแอปพลิเคชันตัวช่วยอำนวยความสะดวก (Accessibility Agent) เพื่อลดช่องว่างทางเทคโนโลยีสำหรับผู้ใช้งานทั่วไป ผู้สูงอายุ และผู้ที่ไม่สันทัดเรื่องไอที ให้สามารถแปลงไอเดียหรือความต้องการในชีวิตประจำวันธรรมดาๆ ให้กลายเป็น **"Prompt (คำสั่ง AI)"** ที่ชัดเจน ครบถ้วน และมีประสิทธิภาพ พร้อมนำไปใช้อาสาสั่งงาน AI ตัวอื่นๆ ได้ทันทีโดยไม่ต้องเรียนรู้วิชาการเขียน Prompt 

---

## 🛠️ สรุปฟีเจอร์และอัปเดตที่เสร็จสมบูรณ์แล้ว

### 1. 💬 โหมดแชทอัจฉริยะ (Chat Assist)
*   **Easy Language Mode (โหมดภาษาง่าย)**: เมื่อเปิดใช้งาน AI จะลดระดับความซับซ้อน ปราศจากคำศัพท์เทคนิคคอมพิวเตอร์ (เช่น AI, Context, Prompt, Constraints) และใช้การอธิบายหรือเปรียบเปรยกับสิ่งใกล้ตัวในชีวิตประจำวันแทน
*   **Tone Selection (การเลือกโทนเสียง)**: ปรับสไตล์คำตอบและ Prompt สำเร็จรูปตามความชอบ ได้แก่ *ทั่วไป, ทางการ, เป็นกันเอง, สนุกสนาน, และกระชับ*
*   **Quick Option Buttons**: ปุ่มทางเลือกคำตอบด่วน (เช่น "เขียนจดหมาย/อีเมล", "สรุปบทความ") ช่วยประหยัดเวลาการพิมพ์สำหรับผู้สูงอายุ
*   **Real-time Recent Chats Sidebar**: แถบประวัติการแชทด้านซ้ายมือที่อัปเดตข้อมูลอัตโนมัติทันทีเมื่อเริ่มพิมพ์ โดยไม่ต้องรีเฟรชหน้าเว็บ พร้อมระบบล้างหน้าจอเมื่อกด New Chat แบบไร้รอยต่อ

### 2. 🩺 Prompt Doctor (วินิจฉัยสุขภาพ Prompt)
*   ระบบประเมินสุขภาพ Prompt ดิบของผู้ใช้ พร้อมคำนวณ **Fit Score (0-100)** แยกกลุ่มวิเคราะห์เป็น **จุดเด่น, จุดด้อย, ข้อเสนอแนะ** และเรียบเรียงเป็น Prompt ที่สมบูรณ์แบบให้อัตโนมัติ (Polished Prompt) ที่มีปุ่มช่วยดำเนินการ ได้แก่ *คัดลอก (Copy), ดาวน์โหลดเป็นไฟล์ .txt, และบันทึกเป็นคลังเทมเพลต*

### 3. 👵 Senior Mode (โหมดหน้าจอใหญ่เพื่อผู้สูงอายุ)
*   ปุ่มสลับการแสดงผลแบบถนอมสายตาที่จะปรับโครงสร้าง UI ขยายขนาดปุ่ม ขนาดฟอนต์ ขนาดสวิตช์ Toggle และกล่องข้อความทั้งหมดให้อ่านง่าย คอนทราสต์ชัดเจน สบายตา

### 4. 🔑 ระบบสมาชิกและแยกสิทธิ์เข้าถึง (User Authentication & Separation - Phase 2)
*   **การสมัครสมาชิกและเข้าสู่ระบบ**: รองรับการแยกข้อมูลประวัติการแชท (Chat History) และเทมเพลตส่วนตัว (Private Templates) แยกตามบัญชีผู้ใช้อย่างเด็ดขาด
*   **ระบบ Token & Hashing ปลอดภัยสูง**: เข้ารหัสพาสเวิร์ดด้วย PBKDF2 Hashing และใช้ Signed JSON-like Token ในการตรวจสอบสิทธิ์โดยไม่ต้องคอมไพล์ไลบรารีภายนอก (Windows Compatible 100%)

### 5. 🏢 แดชบอร์ดสรุปสถิติมัลติเทแนนท์ระดับองค์กร (Stats Dashboard & Multi-tenant - Phase 3)
*   **แยกกลุ่มเทมเพลตองค์กร**: การลงทะเบียนแยกหน่วยงาน/องค์กร (เช่น กฟผ., บริษัท A) สมาชิกในองค์กรเดียวกันสามารถแชร์เทมเพลตเพื่อใช้งานร่วมกันได้ทันที แต่สมาชิกจากต่างองค์กรจะไม่มีวันมองเห็นข้อมูลข้ามกลุ่มกัน
*   **แผงควบคุมสถิติสรุปเชิงลึก (Enterprise Dashboard)**: แสดงตัวเลขสรุปสมาชิกที่เข้ามาใช้งาน, จำนวน Prompt ที่ถูกสร้างขึ้น, คะแนนคุณภาพ Prompt Doctor เฉลี่ย, อัตราความสำเร็จในการใช้งาน (Copy/PDF Export), ชาร์ตแสดงความถี่หมวดหมู่ที่พบบ่อย และตารางเก็บ Log กิจกรรม (Audit Logs) 10 รายการล่าสุด
*   **การพิมพ์รายงานแบบกระดาษ (Export PDF)**: ปุ่มสั่งพิมพ์รายงานสรุปแดชบอร์ดสถิติออกมาเป็นรูปแบบรายงานเอกสาร A4 ปริ้นต์แนบผลงานบริษัทได้อย่างสวยงามและเป็นทางการ

### 6. 🌐 ระบบรองรับสองภาษาและการทดสอบประสิทธิภาพ (Localization & Unit Testing - Phase 4)
*   **รองรับสองภาษา (Multilingual)**: แปลงอินเทอร์เฟสระบบทั้งหมด (หน้าหลัก, แชท, ประวัติ, แดชบอร์ด, Doctor) ให้สามารถสลับภาษาระหว่างไทย-อังกฤษ ได้แบบทันทีด้วย Context API
*   **ระบบการทดสอบ Unit Testing**: เพิ่มโครงสร้างการทดสอบระบบ Backend ด้วย `pytest` ร่วมกับ in-memory Database และ Mock Gemini API เพื่อความเสถียรในการพัฒนาและการอัปเดตในอนาคต

### 7. 📊 การวิเคราะห์แนวโน้มคำสั่งด้วย AI (AI Trend Analysis)
*   **หน้าแดชบอร์ด AI Trends**: เพิ่มฟังก์ชันวิเคราะห์สรุปคำค้นหายอดนิยม ข้อผิดพลาดที่พบบ่อย และแนะนำหัวข้ออบรมพนักงานเพิ่มเติมตามข้อมูลการใช้จริงขององค์กร

### 8. ⚙️ ระบบตั้งค่าผู้ใช้งานแบบครอบคลุม (User Settings & Preferences)
*   **Profile & Accessibility**: จัดการข้อมูลส่วนตัว และปรับแต่งการใช้งานให้เข้ากับแต่ละบุคคล เช่น การปรับขนาดฟอนต์ (Display Font Size) และการเปิด/ปิดโหมดควบคุมด้วยเสียง (Voice Control)
*   **Security & Authentication**: รองรับการตั้งค่าความปลอดภัยขั้นสูง เช่น การสแกนลายนิ้วมือ/ใบหน้า (Biometric Authentication) และการยืนยันตัวตนแบบ 2 ขั้นตอน (2FA)
*   **Notifications Control**: จัดการการแจ้งเตือนต่างๆ ได้อย่างอิสระ ทั้งอีเมลอัปเดตระบบ, Push Notifications บนเดสก์ท็อป และการรับรายงานสรุปการใช้งานประจำสัปดาห์ (Weekly Reports)

### 9. 🕰️ หน้าต่างจัดการประวัติการใช้งาน (Chat History Management)
*   **Search & Filtering**: ค้นหาประวัติการแชทและ Prompt ที่เคยสร้างไว้ได้อย่างรวดเร็ว พร้อมการจัดกลุ่มตามวันที่ (วันนี้ / ก่อนหน้า)
*   **History Export & Actions**: สามารถดูรายละเอียด Prompt ที่เคยสนทนา และมีเมนูจัดการด่วน เช่น คัดลอก (Copy), บันทึกเป็นเทมเพลต (Save), ดาวน์โหลดเป็นข้อความ (.txt) และส่งออกรายงานเป็น PDF ได้ทันทีจากหน้าประวัติ

## 💻 เทคโนโลยีหลักที่ใช้งาน (Tech Stack)

### Backend
*   **FastAPI (Python 3)**: รวดเร็วและแยกส่วน API ชัดเจน
*   **SQLAlchemy + PyMySQL**: เชื่อมต่อฐานข้อมูลเชิงสัมพันธ์
*   **Google Gemini API (`gemini-3.1-flash-lite`)**: ความเร็วสูง ตอบสนองทันใจ รองรับโควตาคำขอใน Free Tier ได้อย่างคุ้มค่า
*   **MySQL Database**: เก็บประวัติบันทึกสถานะการตั้งค่า

### Frontend
*   **Next.js 14 (React / TypeScript)**
*   **Tailwind CSS**: การจัดแต่งสไตล์แบบ Glassmorphism สไตล์แก้วใสและมีแสงสะท้อนพรีเมียม
*   **Responsive Web Design**: รองรับทั้งจอคอมพิวเตอร์ แท็บเล็ต และสมาร์ตโฟน

## 🏃 วิธีการรันระบบเพื่อทดสอบใช้งาน

### 1. การเตรียมระบบ Backend (FastAPI)
1. เข้าไปยังโฟลเดอร์หลังบ้าน:
   ```bash
   cd backend
   ```
2. สร้างและตรวจสอบการตั้งค่าคีย์และฐานข้อมูลในไฟล์ `.env`:
   ```env
   GEMINI_API_KEY=your_gemini_api_key
   DATABASE_URL=mysql+pymysql://user:password@localhost:3306/easyprompt_db
   ```
3. รันเซิร์ฟเวอร์ Backend:
   ```bash
   uvicorn app.main:app --reload
   ```

### 2. การรันระบบ Frontend (Next.js)
1. เข้าไปยังโฟลเดอร์หน้าบ้าน:
   ```bash
   cd frontend
   ```
2. เปิดการทำงานระบบพัฒนา:
   ```bash
   npm run dev
   ```
   *(ระบบจะรันบนพอร์ต http://localhost:3000 หรือ http://localhost:3001 หากพอร์ตแรกถูกจอง)*

### 3. เครื่องมือจัดการฐานข้อมูล (Database Utility Scripts)
ในโฟลเดอร์ `scripts/` จะมีสคริปต์ช่วยเหลือสำหรับการจัดการฐานข้อมูล MySQL (XAMPP) ระหว่างการพัฒนา ได้แก่:
*   `fix_mysql.ps1`: สคริปต์กู้คืนฐานข้อมูล MySQL กรณีโฟลเดอร์ data มีปัญหา
*   `dump_dbs.py`: สคริปต์สำหรับสำรองฐานข้อมูล (Export) เป็นไฟล์ `.sql`
*   `import_dbs.py`: สคริปต์สำหรับนำเข้า (Import) ไฟล์ฐานข้อมูล `.sql` กลับเข้าสู่ระบบ
